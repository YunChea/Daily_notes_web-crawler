# 请求方法
| 方法      | 描述                              |
|---------|---------------------------------|
| GET     | 请求页面，并返回页面内容（最常用）               |
| HEAD    | 类似于GET请求，只不过返回的响应中没有具体内同。用于获取报头 |
| POST    | 大多用于提交表单或上传文件，数据包含在请求体中（次常用）    |
| PUT     | 用客户端传向服务器的数据取代指定文档中的内容          |
| DELETE  | 请求服务器删除指定的页面                    |
| CONNECT | 把服务器当作跳板，让服务器代替客户端访问其他网页        |
| OPTIONS | 允许客户端查看服务器的性能                   |
| TRACE   | 回显服务器收到的请求。 主要用于测试或诊断           |
